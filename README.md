# Online-Ecommerce-Website

This is fully functional e-commerce website, developed using html,css,js, PHP and MySQL. 
It alows usersto browse products,manage their accounts, and experience a dynamic online shopping system.


Feature : 
 1. Product listing and detailed view
 2. Search and browsing functionality
 3. User account management
 4. Update profile information
 5. Add and manage multiple addresses
 6. Secure login and registration system
 7. Dynamic content using PHP
 8. MySQL database integration
 9. Responsive design using Bootstrap
 10. Razorpay integration for payments

Technologies Used:
 1.Frontend: HTML, CSS, Bootstrap, JavaScript
 2.Backend: PHP
 3.Database: MySQL
